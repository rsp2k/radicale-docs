In this tutorial, you will metrics quick start.

### Metrics Quick Start

Enable the metrics endpoint in your configuration:

```ini
[metrics]
enabled = True
endpoint = /.metrics
```text
Access metrics at `http://localhost:5232/.metrics`

```bash
curl http://localhost:5232/.metrics
```text
